# Superstore Sales Dashboard

## Project Overview
This is an **interactive Power BI dashboard** for Superstore sales data analysis (2015–2018).  
It provides insights into overall sales performance, regional comparison, product sub-categories, and customer segments.

## Features
- **KPI Section**: Total Sales, Total Orders, Total Customers, Average Sales per Order
- **Sales Trend Over Time**: Line chart showing yearly/monthly sales
- **Sales by Region**: Compare sales performance across regions
- **Sales by Sub-Category**: Identify best and worst selling products
- **Sales by Segment**: Analyze performance by customer segment
- **Interactive Slicers**: Filter by Region, Segment, and Order Date
- Fully interactive: all charts and KPIs respond to slicers

## Tools & Technologies
- **Power BI Desktop**
- **DAX Measures** for KPI calculations

## Insights
- West region has the highest sales; South has the lowest
- Phones are the top-selling sub-category; Machines are lowest
- Consumer segment drives the most revenue
- Sales dipped in 2016 but grew steadily in 2017–2018

## File
- `Superstore_Sales_Dashboard.pbix` — Power BI dashboard file

## How to Use
1. Open `Superstore_Sales_Dashboard.pbix` in Power BI Desktop
2. Use slicers to filter by Region, Segment, and Date
3. Explore charts and KPIs to gain business insights
